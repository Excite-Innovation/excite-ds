# Upload

The Upload component provides a simple and flexible way to upload files in React applications. It supports direct uploads to S3-compatible storage providers and can be customized to fit different workflows and user experiences.

## Features

- Simple file uploads
- Drag-and-drop support
- Direct uploads to S3-compatible storage
- Customizable UI and behavior
- TypeScript support

## Frequently Asked Questions

<Accordions type="single">
  <Accordion title="What file types are supported?">
    The Upload component supports any file type. You can restrict accepted file types using the <code>accept</code> prop.
  </Accordion>

  <Accordion title="Can I upload files directly to S3?">
    Yes. The component is designed to work with Amazon S3 and other S3-compatible storage providers, enabling secure direct uploads.
  </Accordion>

  <Accordion title="Does it support drag and drop?">
    Yes. Users can either drag and drop files into the upload area or select files manually using the file picker.
  </Accordion>

  <Accordion title="Can I customize the upload UI?">
    Yes. The component is fully customizable and can be styled to match your application's design system.
  </Accordion>
</Accordions>

# Excite! Design System

Excite! Design System is a shared UI foundation for building consistent, scalable, and accessible products across Excite Innovation.

## Purpose

This system exists to:
- Keep UI consistent across all products
- Speed up product development
- Reduce duplicated design and engineering work
- Ensure accessibility by default

## What’s included

- **Foundations**: color, typography, spacing, elevation
- **Components**: buttons, inputs, dialogs, navigation
- **Patterns**: forms, dashboards, data tables
- **Tokens**: design variables used across apps
